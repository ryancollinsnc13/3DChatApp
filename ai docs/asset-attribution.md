# Asset Attribution

## Generated Local Game Asset Library

The following GLB files are generated by `scripts/build-assets.mjs` and stored in `public/models/game/`.

- `character-base.glb`
- `house-kit.glb`
- `room-shell.glb`
- `furniture-kit.glb`
- `decor-plant.glb`

These are original project assets authored through Three.js geometry and exported with `GLTFExporter`. They are intended to be the reusable local asset library for the prototype, similar to a game character/environment kit. Runtime code recolors and toggles parts of these models for avatar and house customization.

The current `character-base.glb` is a part-based avatar rig with named meshes for head, skin, ears, hair cap, bob sides, spiky hair, swoop lock, short fringe, brows, eyes, highlights, blush, mouth, glasses, outfit, arms, hands, legs, and shoes. These parts are controlled by the avatar builder sliders and style tabs.
